01

What is Hermes?

Hermes is a system for tracking employee activity in warehouses and departments at Yandex Market. With this system, managers can view statistics on employees’ work shifts, their occupancy, the number of hours worked, days off, sick leave, and plan work accordingly based on this information.

02

The problem

The issue in the current utilization of numerous different programs at work, which partially or fully duplicate each other. The plan is to make Hermes the primary tool for warehouse management.

03

The purpose

We need to assess its current functionality, gather feedback from supervisors, take their input into account, brainstorm ways to improve the interface, add essential features, and carry out a redesign.

04

Competitive analysis

Together with the team, we conducted an analysis of six different services to better understand trends in this field, as well as to take into account their advantages and avoid repeating mistakes.

hermes — competitive analysis
05

Hypotheses

After the research stage, we identified several hypotheses on how to solve this problem:

  1. 01We suggest that if we display line numbering in the table, it will be easier for users to navigate through the list of employees.
  2. 02We suggest that if we show notifications about company news, it will be easier for users to stay informed about everything, and they are less likely to miss anything.
  3. 03We suggest that if we add a structured profile card for employees with personal and professional information, it will be easier for users to find all the necessary data about the employees.
  4. 04We suggest that if we allow editing of employee statuses, users will see up-to-date information in the table and won’t miss anything.
  5. 05We suggest that if we display employee performance and efficiency in the profile, users will find it easier and quicker to make decisions about bonuses.
06

User stories

After analyzing the interviews, we formulated several user stories:

  1. 01As a brigade leader, I want to see line numbering in the table so that it’s easier to navigate through the list of employees.
  2. 02As a packing area manager, I want to receive notifications about company news so that I don’t miss important information.
  3. 03As a brigade leader, I want to see a structured profile card for each team member with personal and professional information so that I can quickly find the necessary data.
  4. 04As a brigade leader, I want to see a structured profile card for each team member with personal and professional information so that I can quickly find the necessary data.
  5. 05As a brigade leader, I want to see employee ratings to make decisions about bonuses.
07

Tests & Prototyping

We gave real respondents with a clickable prototype of the interface along with clear guidelines and tasks, monitoring the time spent by the respondent on each task.

hermes — prototype
08

New features

Based on the compiled User Stories and the research stage, we came up with several new features:

1

Notifications

Based on the interviews, it was found that brigade leaders often miss important information, which significantly hinders the workflow. The solution was to create a notification system so that users wouldn’t forget important tasks.

2

Changing employee status

Based on the interviews, it was found that brigade leaders often miss important information, which significantly hinders the workflow. The solution was to create a notification system so that users wouldn’t forget important tasks.

3

Employee profile and statistics

Previously, all information about employees was stored in separate paper folders. If a brigade leader needed any information about an employee, they had to ask other colleagues or search for it themselves. Now all information about the employee is located in one place, and the statistics section will help make the right decision about bonuses.

4

Updated table design

We changed the color scheme and added intuitive icons, and thanks to the green dotted line, the user will never lose sight of today. Also, to make it easier to navigate the table, we increased its scale and added a search by name. Now the user won’t get confused with all the values in the table, can use the search to quickly find the employee they need, and won’t strain their eyes when using it.
